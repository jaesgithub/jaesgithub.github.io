Types of Semantic Elements

  • <header>(Semantic Element, a container fo introductory content)
  • <nav>(Semantic Element, a container for a set of navigation links
  • <section>(Semantic Element, defines sections in a document)
  • <footer>(Semantic Element, container for the bottom of the webpage)

A <header> element typically contains:

  • One more heading elements
  • Logo or icon
