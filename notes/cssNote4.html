Border & Padding & Margin

Notes

border-style

  • CSS Property
  • Sets a style for the border

    • Values of border-style

    • none
    • dashed
    • double
    • ridge
    • dotted
    • solid
    • groove
    • inset

border-width

  • CSS Property
  • Sets a width for the border

    • Values of border-width

    • medium
    • thin
    • thick
    • _px

border-color

  • CSS Property
  • Sets a colour for the border

Padding

  • CSS Property
  • Used to give space inside the border

    • Properties of Padding

    • padding-top
    • padding-right
    • padding-bottom
    • padding-left

Margin

  • CSS Property
  • Used to give space outside the border

    • Properties of Margin

    • margin-top
    • margin-right
    • margin-bottom
    • margin-left
