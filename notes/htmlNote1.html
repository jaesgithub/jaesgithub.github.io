HTML(HyperText Markup Language)

  • Language for creating Web pages
  • First line of code should always start by telling it is a HTML file
  • There are only two elements inside <html>:

Elements

  • Individual pieces of a HTML document
  • Tags(Tells where an element begins and ends)
  • The first element in the HTML(The entire content of the HTML file is inside it)
  • <title>(Title that is displayed on the top of the page)
  • There are only two elements inside <html>:

Two elements inside of HTML

  • <head>(HTML Element, Where you put information that isn't visible in the website
  • <body>(HTML Element, Main content od the page, containing text, images, tables, etc.)

BODY Elements

  • <p>(Paragraph, BODY Element, Creates a paragraph)
  • <h1>(Header, BODY Element, Used as the header of the document, The size decreases as the number change from 1 to 6
  • <ul>(Unlisted List, BODY Element, Creates an empty unordered list)
  • <li>(List Item, BODY Element, Creates one bullet)
  • <ol>(Ordered List, BODY Element, Creates an empty ordered list)

Writing Code

  1. <!DOCTYPE html>
  2. <html>
  3. </html>
