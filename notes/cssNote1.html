CSS

  • Website Language
  • Language for stylizing HTML files

CSS Rules

  • Selector(Points to the HTML element you want to style)
    • Points to the HTML element you want to style
  • Property(Stylistic feature you want to change
    • Stylistic feature you want to change
  • Value(How you want to change its stylistic feature)

    • i.e. p(selector) {font-size:(property) 12px}(value)

    {font-size: 12px} being the declaration

Common CSS Properties

  • font-size
    • Used to size the text of a tag
  • line-height
    • Used to size the height of the text lines
  • color
    • Used to set the text colour of an element
