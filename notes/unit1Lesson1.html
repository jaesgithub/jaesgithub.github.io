Constant and Variable

  • Constant
    • Something that cannot be changed, and is constant, written as "let"
  • Variable
    • Something that can be changed, written as "var"

Type Inferences

  • Used to make assumptions about the type based on the value assigned to the constant or variable

  • Boolean
    • Represents texts questioning "true or false?"

  • Double
    • Represents numbers with decimals

  • Integer
    • Represents numbers that are not decimals, only integers

  • String
    • Represents texts and words written and must include double quotations
