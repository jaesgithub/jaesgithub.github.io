<img>

  • <img> (Defines image in a HTML page)
  • src(Source, ,img. Attribute, Specifies the URL of an image)
  • height & width(,img. Attribute, used to resize the image in pixels)
  • you can specify only one height or width and the aspect ratio will be adjusted

Two different ways to use src

  • Using URL of Image(Method #1)
  • Saving the Image File(Method #2)

  • first, the image should be saved in the folder containing the HTML code

Advantages of using Method #2

  • if the image is deleted in the Internet, the image won't be displayed
