Inheritance

  • Unlike structures, class can have hierarchial relationships
  • The parent class is superclass, and child class is subclass
  • The subclass inherit properties from superclasses, but also augment or replace the implementation of superclass methods

    • Defining a Base Class

    • A class that does not inherit from superclass os known as base class
    • A base class defines common characteristics, for properties or methods

    Create a Subclass

    • The act of basing a new class on an existing class
    • Inherits characteristics such as properties or methods from superclass in order to make more specific

Override Methods and Properties

  • In subclass, each one can provide its own custom implementation of a property or method
  • To override, prefix new definition with "override"
