Instances

  • To use type, must create an instance of it, a process called initialization
  • After initialization, each instance inherits all properties and features of structures

Initializers

  • All Swift types come with initializers, similar to functions returns new instance of type
  • Default initializer is "init( )"

Default Values

Memberwise Initializers

  • Used sometimes in order to create new instance with non-default values
  • When defining a new structure, Swift creates a special initializer, called memberwise initializer
  • The memberwise initializer contains all properties of structure
  • Allows you to set initial values for each property of the new instance
  • To call a memberwise

Custom Initializers

  • Customize initialization process by defining your own initializer

Instance Methods

  • Functions that called on specific instances of type
  • Provides ways to access, modify properties of structures, they add functionality that relates to instance's purposes

Mutating Methods

  • Mutates (updates) property values of structure within instance method
